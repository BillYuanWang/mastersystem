# 佳美舞蹈排课系统

这是一个本地网站系统。推荐双击 `打开 Master Dance.command` 使用，它会自动打开浏览器并读取同一文件夹里的 `courses.csv`。

不要直接双击 `index.html` 作为日常入口：浏览器会因为本地文件安全限制，拦截网页自动读取旁边的 `courses.csv`，页面可能显示为空。

- `课程表`：按星期、教室、年龄段、课程类型、老师筛选，并可打印当前视图。
- 课程表支持 70%-180% 缩放，按住 Command/Ctrl 后滚轮也可以缩放。
- `课程设置`：添加、编辑、复制、删除课程，也可以筛选和排序课程列表。
- `学生`：自动汇总所有已报名孩子，一个孩子一列，显示已报名课程、报名方式和预估应付金额；课程块可点开只读查看。
- 顶部 `外观` 可切换 `跟随系统`、`浅色`、`深色`。
- 每门课可设置组课/私课、单期价格、按期价格，以及报名学生名单。
- 每门课可设置起始周和结束周，系统会按课程星期自动生成上课周次；不上的周次可以点 `×` 扣掉，并写入 CSV 的 `停课日期`。
- 报名学生支持 `按期` 和 `按N次` 两种方式；课程表悬停会按不同颜色显示学生。
- 课程数据保存在同一文件夹里的 `courses.csv`。通过 `打开 Master Dance.command` 进入时，页面打开就会显示 CSV 数据，添加、编辑、删除都会自动写回这个 CSV。
- `下载CSV` 保留给需要另存或发给别人的时候使用。
- `CSV 备用` 里保留 `重新授权`、`新建CSV`、`导入替换`，主要用于浏览器权限异常或特殊维护，不是日常入口。
- CSV 里的 `报名学生` 格式为 `姓名|按期; 姓名|按N次`。
- 浏览器只保存课程表缩放偏好，以及可选的 CSV 文件授权句柄；课程内容不依赖浏览器存储。
